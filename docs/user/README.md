# Global Dashboard — User Guide

## What is this?

Global Dashboard is a personal financial intelligence platform that combines 16 analytical tools in one interface. It pulls live market data, runs quantitative models, and surfaces actionable signals across options, technicals, fundamentals, sentiment, institutional ownership, and AI research.

## Quick Start

1. Type a ticker in the top bar (e.g. `AAPL`, `NVDA`, `SPY`)
2. Select a timeframe (1H → 1Y)
3. Navigate to any feature in the left sidebar

## Features at a Glance

| Feature | Best for |
|---|---|
| **Reversal Scanner** | Finding stocks about to change direction |
| **Technical Analysis** | Chart patterns, momentum, support/resistance |
| **Fundamental Analysis** | Valuation, growth, financial health |
| **Options Analysis** | Options chain, Greeks, IV skew, unusual activity |
| **Earnings Calendar** | Upcoming earnings with expected move sizing |
| **Fear & Greed** | Market mood — are investors fearful or greedy? |
| **News Sentiment** | AI-scored sentiment on stock news |
| **Insider Tracker** | What company executives are buying/selling |
| **Congress Tracker** | What politicians are trading |
| **13F Holdings** | What institutions (hedge funds, mutual funds) own |
| **Stock Rankings** | Multi-factor ranked stock picks |
| **Multi-Factor Screener** | Find stocks matching your criteria |
| **Backtest & RL** | Track how well the dashboard's signals actually predict prices |
| **AI Research Agent** | Ask Claude to research any stock |
| **Portfolio Tracker** | Track your positions and P&L |
| **Alerts & Watchlist** | Price alerts and watchlist |

## Sidebar Navigation

The sidebar groups features by category:
- **Analysis** — single-stock deep dives
- **Market** — macro and market-wide views
- **Smart Data** — alternative data (insiders, congress, institutions)
- **Rankings** — ranked lists and screeners
- **AI** — AI-powered research and backtesting
- **Portfolio** — personal tracking

## Tips

- The ticker bar at the top applies globally — change it once and all tabs update
- Most data is cached for 5–30 minutes to avoid rate limits
- Options data comes from Yahoo Finance — it can be slow during market open
- The AI agent requires a Groq or Anthropic API key to function
