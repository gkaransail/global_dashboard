# Technical Analysis — User Guide

## What does it do?
Gives you the standard trader's toolkit — momentum indicators, chart patterns, and support/resistance levels — computed automatically for any stock.

## Tabs

### Indicators
Live values and recent history for the most-used technical indicators:

| Indicator | What to look for |
|---|---|
| **RSI** | > 70 = overbought (caution), < 30 = oversold (opportunity) |
| **MACD** | Histogram turning positive = momentum shift up; negative = shift down |
| **Bollinger Bands** | Price touching upper band = extended; lower band = compressed |
| **EMA 20/50/200** | Price above all three = strong uptrend; below all = downtrend |
| **Stochastic** | > 80 = overbought, < 20 = oversold |
| **ATR** | Higher ATR = more volatile stock, size positions accordingly |

### Chart Patterns
Automatically detects classic patterns like Head and Shoulders, Double Top/Bottom, Cup and Handle, Flags, Wedges. Each pattern shows:
- **Confidence** — how clearly the pattern formed
- **Price Target** — where the pattern projects to
- **Invalidation** — the price where the pattern fails

### Support & Resistance
Key price levels where the stock has historically found buyers (support) or sellers (resistance). These are derived from:
- Swing highs and lows
- High-volume price clusters
- 52-week high and low
- Round numbers

Use these as entry/exit points and stop-loss placement guides.

### Screener
Scans 50 popular stocks for specific technical conditions. Filter by conditions like "RSI oversold", "above 200 EMA", "MACD crossover" and get a ranked list of matches.

## How to use it
1. Start with **Indicators** to assess current momentum (is RSI coming off 30? MACD crossing positive?)
2. Check **Patterns** for structure — is there a clear setup with a defined target and stop?
3. Use **Levels** to identify entry zones and stop-loss placement
4. The **Screener** is best for finding opportunities across the market quickly

## Tip
Technical analysis works best when multiple indicators agree. A single RSI oversold reading is weak — RSI oversold + MACD bullish cross + price at support = much stronger setup.
